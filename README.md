# SoundWaveApp

## What is SoundWave?

A Social Music Connection and Discussion application

* Allows users to match with others who have similar music preferences 

* Allows for discussion across a variety of music topics


## App Goals

* Users can upload their music library

* Users can find other users with overlapping music taste

* Users can search artists, albums, or songs 

* Users can post about artists, albums, or songs


## Tech Stack

Frontend :

* React VITE, VS Code

Backend :

* Java, Spring Boot, IntelliJ, Postgres

* ClickUp, Notion
